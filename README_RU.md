# Prompt-generation 🎨 Краткое руководство

Prompt-generation — это лёгкое настольное приложение для работы с промптами в системах генерации изображений с помощью ИИ, таких как Stable Diffusion, Midjourney, FLUX и других.

Вместо использования неудобных текстовых файлов или таблиц программа предоставляет интуитивно понятный интерфейс, позволяющий удобно организовывать, редактировать и генерировать промпты без программирования.

## 🚀 Быстрый старт

### Шаг 1. Загрузка и запуск

**Для большинства пользователей (рекомендуется)**

Скачайте последний `.zip` архив со страницы GitHub Releases, распакуйте его и запустите `PromptGenerator.exe`.

**Для разработчиков**

Требуется Python 3.8 или выше.

```bash
pip install PyQt6
python main.py
```

### Шаг 2. Выбор тегов и настройка весов

#### Загрузка библиотеки

Приложение включает встроенную библиотеку промптов по умолчанию.

Также можно загрузить собственную JSON-библиотеку через:

`File → Load Library...`

#### Выбор тегов

Просматривайте категории и выбирайте нужные теги одним щелчком мыши.

#### Настройка веса

Нажмите значок ⚙ рядом с тегом для быстрой настройки его веса.

Пример:

```text
(masterpiece:1.3)
```

#### Пользовательские промпты

Используйте поле «Custom Prompts» для добавления временных идей или дополнительных модификаторов.

### Шаг 3. Режим вдохновения

Если вам нужны новые идеи, нажмите:

🎲 Global Random

Программа создаст случайную комбинацию в отдельном блоке вдохновения, не изменяя текущий набор выбранных тегов.

Если результат вас устраивает, нажмите:

✨ Apply Inspiration

чтобы мгновенно применить его.

### Шаг 4. Генерация и копирование

Нажмите:

🚀 Generate Prompt

После этого приложение покажет:

* Локализованный предварительный просмотр для проверки
* Чистый английский промпт для генерации изображений

Затем нажмите:

📋 Copy English

и вставьте результат в ваш любимый инструмент генерации изображений.

## 💡 Полезные советы

### Визуальный редактор библиотек

Нажмите:

```text
Ctrl + E
```

чтобы открыть визуальный редактор библиотек.

Вы можете создавать категории, организовывать промпты и редактировать библиотеки через удобный интерфейс, похожий на электронную таблицу, без ручного редактирования JSON-файлов.

### Персонализация

Приложение поддерживает:

* Многоязычный интерфейс
* Мгновенное переключение языков
* Три встроенные темы

  * Dark Cyber
  * Light Minimal
  * Dracula Purple
