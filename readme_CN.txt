# Prompt-generation

🚀 **一个数据驱动、高度模块化的 AI 提示词管理与生成桌面应用。**

本项目基于 Python 与 PyQt6 开发，拥有响应式界面与现代化暗色主题设计。
整体架构严格遵循 **关注点分离（Separation of Concerns, SoC）** 思想，实现了：

* 核心解析引擎
* UI 渲染层
* 配置数据层

三者完全解耦。

它能够帮助用户更加高效地管理、随机抽取、调整与生成 AI 绘图提示词，适用于：

* Stable Diffusion
* Midjourney
* Flux
* 以及其他 AI 图像生成工具

---

# ✨ 功能特性

## 📂 数据驱动动态 UI

无需硬编码。

只需编辑 JSON 配置文件，系统即可自动解析并动态生成多层级 UI 树结构。

支持：

* 动态扩展
* 模块化分类
* 自定义提示词库

---

## 🌊 响应式流式布局

内置自定义 Flow Layout（流式布局）算法。

标签按钮会根据窗口宽度自动换行，完美适配：

* 小屏窗口
* 超宽屏
* 不同 DPI 缩放

---

## ⚙️ 无缝权重调整

点击标签旁边的 `⚙` 图标，即可弹出无边框悬浮权重调节面板。

支持：

* 键盘精准输入
* 小数权重控制
* 实时格式化

示例：

```text
(tag:1.2)
```

---

## 🎲 智能随机抽取

支持：

* 模块级随机抽取
* 全局随机生成

算法具备分类感知能力，可自动避免逻辑冲突标签同时出现。

例如：

* ✅ `sunrise`
* ❌ `sunrise + midnight`

---

## 🌐 双栏提示词输出

底部输出区域采用双栏布局：

| 左侧         | 右侧         |
| ---------- | ---------- |
| 原语言预览（带权重） | 纯英文 Prompt |

方便用户：

* 校对
* 调整
* 复制使用

---

## 🎨 现代化暗黑主题

深度定制 QSS 暗色赛博风主题。

特点：

* 长时间使用更护眼
* 高对比度
* 现代化视觉风格
* 更适合 AI 创作环境

---

# 🛠️ 安装方法

请确保系统已安装：

* Python 3.8 或更高版本

---

## 1️⃣ 克隆仓库

```bash
git clone https://github.com/YOUR_USERNAME/Prompt-generation.git
cd Prompt-generation
```

---

## 2️⃣ 安装依赖

本项目非常轻量，仅依赖 PyQt6。

```bash
pip install PyQt6
```

---

## 3️⃣ 运行程序

```bash
python main.py
```

---

# 🚀 使用方法

## 1️⃣ 选择标签

浏览分类并点击需要使用的标签。

被选中的标签会高亮显示为蓝色。

---

## 2️⃣ 调整权重

点击标签右侧的小齿轮：

```text
⚙
```

即可打开权重调整面板。

支持：

* 手动输入
* 小数权重
* 上下箭头调节
* 自动格式化

示例：

```text
(tag:1.2)
```

---

## 3️⃣ 随机抽取

### 模块随机

点击：

```text
🎲 随机抽取
```

仅会在当前模块内随机抽取标签。

---

### 全局随机

点击：

```text
🎲 全局一键随机
```

将在所有已展开模块中随机生成标签。

> 已折叠模块会被自动忽略。

---

## 4️⃣ 生成提示词

点击：

```text
🚀 生成提示词
```

底部输出区域会立即生成最终 Prompt。

复制右侧英文 Prompt，即可直接粘贴到 AI 绘图工具中使用。

---

# 🏗️ 项目结构

```text
Prompt-generation/
├── main.py
│
├── ui/
│   ├── main_window.py
│   ├── flow_layout.py
│   └── components.py
│
├── core/
│   └── parser.py
│
├── data/
│   └── example_landscape.json
│
└── assets/
    └── styles.qss
```

---

# 📖 结构说明

| 文件 / 目录             | 说明         |
| ------------------- | ---------- |
| `main.py`           | 程序入口与初始化   |
| `ui/`               | UI 渲染模块    |
| `flow_layout.py`    | 自定义响应式流布局  |
| `components.py`     | 可复用组件      |
| `core/parser.py`    | JSON 动态解析器 |
| `data/`             | 用户提示词数据目录  |
| `assets/styles.qss` | 全局暗色主题样式   |
