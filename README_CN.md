Prompt-generation 🎨 极简使用指南
这款轻量级桌面工具专为 AI 绘画（如 Stable Diffusion, Midjourney, FLUX 等）设计。它用直观的“点击式”界面彻底替代了混乱的文本文档和 Excel，让你零代码、高效率地管理和生成提示词（Prompts）。

🚀 快速上手步骤
第一步：获取与运行
普通用户（免安装）：直接在 GitHub Releases 页面下载最新的 .zip 压缩包，解压后双击 PromptGenerator.exe 即可直接使用。

开发者：需 Python 3.8+ 环境，运行 pip install PyQt6 后执行 python main.py。

第二步：点选标签与调重
加载词库：软件默认自带基础词库。你也可以通过菜单栏 File → Load Library... 加载自己的 JSON 词库。

点选组合：在界面中浏览不同的分类模块，直接点击你想要的标签（Tag）。

调整权重：如果需要强化某个词，点击该标签旁边的 齿轮图标 (⚙)，即可快捷调整数值（例如自动生成 (masterpiece:1.3)）。

临时补充：如果有临时冒出来的想法，直接打字输入到底部的 "Custom Prompts"（自定义提示词） 文本框中。

第三步：无损灵感抽卡（特色功能）
没灵感时，点击 🎲 全局一键随机 (Global Random)。

软件会在独立的“灵感框”内生成一套随机组合，这完全不会破坏或覆盖你当前已经选好的词。

看到满意的组合，点击 ✨ 应用此灵感 (Apply Inspiration)，将其一键替换到你的工作区。

第四步：一键生成与复制
点击 🚀 生成提示词 (Generate Prompt)，界面会分两栏展示：一栏是方便你检查的母语预览，另一栏是喂给 AI 的纯英文提示词。

点击右下角的 📋 复制英文 (Copy English)，直接粘贴到你的 AI 绘画工具中开始出图！

💡 进阶实用技巧
可视化编辑 (Pro Tip)：按下 Ctrl+E 即可打开词库编辑器。你可以像填表格一样，直观地添加自己的专属分类和常用标签，彻底告别手动修改 JSON 文件的痛苦。

个性化界面：在菜单栏中，你可以一键无缝切换界面语言（支持中、英、日等多国语言），并随意更换内置的 3 套精美皮肤（暗黑科技、明亮简约、德古拉紫）。
